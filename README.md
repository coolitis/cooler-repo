## üåê Installation Instructions (Kodi)

1. Open **Kodi**.  
2. Go to **Settings ‚Üí File Manager ‚Üí Add Source**.  
3. Select **<None>**, then enter the repository URL:
https://coolitis.github.io/cooler-repo/

4. Name it **Cooler Repo** and click **OK**.  
5. Return to the main menu ‚Üí **Add-ons ‚Üí Install from zip file**.  
6. Choose **Cooler Repo** ‚Üí install `repository.cooler-repo.zip`.  
7. Then go to **Install from repository ‚Üí Cooler Repo** to browse and install addons.
